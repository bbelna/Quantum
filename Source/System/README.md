## Structure
| Directory | Description |
| ----------- | ----------- |
| `.\Boot\` | Bootloaders. |
| `.\Coordinator\` | The system coordinator. |
| `.\Drivers\` | Drivers. |
| `.\FileSystems\` | File system services. |
| `.\Kernel\` | The kernel. |
| `.\User\` | User-mode system sources and startup glue. |
