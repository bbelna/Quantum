## Overview
This folder contains the Quantum kernel implementation.

## Structure
| Directory | Description |
| ----------- | ----------- |
| `.\` | Core arch-agnostic kernel sources. |
| `.\Arch\` | Arch-specific kernel sources. |
| `.\Handlers\` | Kernel event handlers. |
| `.\Helpers\` | Generic helper classes. |
| `.\Include\` | Kernel header files. |
| `.\Runtime\` | C++ runtime code. |
| `.\Tests\` | Kernel test cases. |
